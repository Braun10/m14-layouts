Welcome to Hierarhcy Exercise 1

This exercise challenges you to use D3 Layout Functions to build a Treemap. In your js/main.js file, do the following:

  • Create hierarchical data structure & treemap function
    • Nest your data by region by using the d3.nest function.
    • Place your data in a single hierarchical object by using the d3.hierarchy() function
  • Create an ordinal color scale
    • Get a list of your regions from your nested data object. This will be the domain of your ordinal scale.
    • Use d3.scaleOrdinal() to create an ordinal scale. Your domainwill be your region names and your range should be the colors stored in the variable d3.schemeCategory10
  • Write your draw function to bind data, and position elements.
    • Redefine which value you want to visualize in your data by using the .sum() method of your root object.
    • Perform a data-join between you selection of elements with class node and the array of objects returned by root.leaves()
